# modmail-bot

A Discord modmail bot made with Discord.js

## Installation

Make sure to install the dependencies:

```sh
# yarn
yarn install

# npm
npm install
```

## Environment Variables

Copy the `.env.example` file into `.env`:

```sh
cp .env.example .env
```

Update the values accordingly.

## Usage

Start the bot:

```sh
# yarn
yarn dev

# npm
npm run dev
```
