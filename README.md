# modmail-bot

A Discord modmail bot made with Discord.js

## Installation

Make sure to install the dependencies:

```sh
# yarn
yarn install

# npm
npm install
```

## Environment Variables

Copy the `.env.example` file into `.env`:

```sh
cp .env.example .env
```

Update the values accordingly.

## Development

Start the bot:

```sh
# yarn
yarn dev

# npm
npm run dev
```

## Production

Build the bot for production:

```sh
# yarn
yarn build

# npm
npm run build
```

Start the bot:

```sh
# yarn
yarn start

# npm
npm run start
```
